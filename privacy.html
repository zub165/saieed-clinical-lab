3. How We Use Your Information

3.1 Primary Uses

  • Account Management: User authentication, profile management, account recovery
  • Medical Records: Patient data management, test result storage, medical history tracking
  • Appointment Scheduling: Calendar integration, appointment reminders, scheduling management
  • Laboratory Operations: Test ordering, result processing, report generation
  • Billing & Payments: Payment processing, invoice generation, financial records
  • Data Synchronization: Hybrid storage system (local + cloud backup)

3.2 Secondary Uses

  • App Improvement: Performance optimization, feature development
  • Customer Support: Technical assistance, issue resolution
  • Compliance: Regulatory requirements, audit trails
  • Security: Fraud prevention, unauthorized access detection

4. Data Storage & Security

🔐 Security Measures

  • Encryption: All data encrypted in transit (HTTPS) and at rest (AES-256)
  • Authentication: Multi-factor authentication support
  • Access Control: Role-based permissions and secure API endpoints
  • Local Storage: SQLite database with encryption
  • Backend Security: Django REST Framework with token authentication
  • Regular Backups: Automated backup systems with encryption

4.1 Hybrid Storage System

Our app uses a hybrid storage approach:

  • Local Storage: SQLite database on your device for offline access
  • Cloud Sync: Secure backend synchronization when online
  • Data Priority: Local data takes precedence, with cloud as backup
  • Conflict Resolution: Automatic conflict resolution with user notification

To permanently delete your account and all associated data:

  1. Send an email to: zub165@yahoo.com
  2. Subject line: "Account Deletion Request - SAEED Lab App"
  3. Include your registered email address and user ID
